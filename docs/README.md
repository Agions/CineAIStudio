# 欢迎使用 ClipFlowCut

ClipFlowCut 是专业的 **AI 驱动** 视频创作桌面应用，区别于传统手动剪辑软件，ClipFlowCut 通过 AI 实现自动化创作。

| 🤖 AI 视频解说 | 原视频 + AI 配音 + 动态字幕 |
