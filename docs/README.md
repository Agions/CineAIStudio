# 欢迎使用 Voxplore

Voxplore 是专业的 **AI 第一人称视频解说** 桌面应用。上传任意视频，AI 自动代入画面主角视角，生成"我"的口吻配音解说 + 电影级字幕 + 精准同步 ASS 字幕。

## 核心工作流

| 步骤 | 说明 |
|------|------|
| 📹 上传视频 | 支持 MP4 / MOV / AVI / MKV / WebM |
| 🧠 AI 场景理解 | Qwen2.5-VL 逐帧分析画面，识别主角视角 |
| ✍️ 生成解说稿 | DeepSeek-V4 代入"我"的第一人称视角 |
| 🎙️ 情感配音 | Edge-TTS / F5-TTS 高保真合成 |
| ✨ 合成输出 | MP4 成品 / 原生剪映草稿 JSON |

---

## 主要特性

| 特性 | 说明 |
|------|------|
| **第一人称代入** | Qwen2.5-VL 识别画面主角，DeepSeek-V4 代入"我"视角 |
| **5 种情感风格** | 治愈 / 悬疑 / 励志 / 怀旧 / 浪漫 |
| **精准 ASS 字幕** | TTS word-level timing，50ms 以内同步精度 |
| **多格式导出** | H.264/H.265 MP4 或原生剪映草稿 JSON |
| **隐私优先** | 视频不上传云端，API Key 自持 |
| **开源免费** | MIT 协议，代码完全透明 |
