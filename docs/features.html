功能介绍 ​

VideoForge 提供六大 AI 创作模式，覆盖从影视解说到社交媒体分发的全场景。

🎬 剧情分析 ​

智能分析视频的叙事结构、情感曲线和剧情发展，自动生成基于故事的剪辑建议。

核心能力 ​

  • 场景分割 — 自动识别场景边界，生成场景列表
  • 情感分析 — 分析每个场景的情感倾向（喜怒哀乐）
  • 叙事提取 — 提取关键情节和高光时刻
  • 剪辑建议 — 基于叙事结构推荐剪辑点

适用场景 ​

  • 📽️ 电影解说 / 影评视频
  • 📷 Vlog 整理与叙事重构
  • 🎞️ 纪录片素材整理
  • 📚 课程视频结构化

界面位置 ​

📸 界面截图位置：app/ui/main/pages/ai_video_creator.py → 模式选择器 → 剧情分析模式

🎙️ AI 视频解说 ​

将原始视频自动转化为带 AI 配音和动态字幕的解说视频。

工作流程 ​

视频导入 → 场景分析 → 文案生成 → 配音合成 → 字幕制作 → 视频合成 → 导出

核心能力 ​

步骤技术说明
场景分析PySceneDetect / Gemini识别视频结构和内容
文案生成GPT-4o / Claude生成自然流畅的解说词
配音合成Edge TTS / OpenAI TTS支持多音色、多语言
字幕生成Whisper语音转文字 + 时间轴
视频合成FFmpeg音画合成 + 字幕叠加

适用场景 ​

  • 🎓 在线课程录制
  • 📦 产品介绍视频
  • 📰 新闻/纪录片配音
  • 🎮 游戏解说

界面位置 ​

📸 界面截图位置：app/ui/main/pages/ai_video_creator.py → 模式选择器 → AI 解说模式

参数配置 ​

参数说明推荐值
解说风格专业 / 轻松 / 新闻专业
语速0.8x ~ 1.5x1.0x
音调低沉 / 正常 / 高亢正常
字幕样式电影 / 卡片 / 底部电影

🎵 AI 智能混剪 ​

将多个素材智能混剪成节奏感强的视频，自动完成节拍匹配和转场合成。

工作流程 ​

多素材导入 → 节拍检测 (BPM) → 素材排序 → 剪辑点生成 → 转场合成 → 导出

核心能力 ​

  • BPM 检测 — 自动检测音频/视频的节拍，精确到毫秒
  • 智能排序 — 根据节拍和内容关联性自动排序素材
  • 自动转场 — AI 生成适合节奏的转场效果（叠化、闪白、节奏切）
  • 多轨混音 — 支持背景音乐 + 原音混合

适用场景 ​

  • 🎵 音乐 MV / 混剪视频
  • ⚽ 体育精彩集锦
  • 🎮 游戏高光时刻
  • 📸 旅行/生活照片集

界面位置 ​

📸 界面截图位置：app/ui/main/pages/ai_video_creator.py → 模式选择器 → 智能混剪模式

独白生成生成符合情感的内心独白
情感配音使用更柔和/更有感染力的声音

📱 AI 短视频切片 ​

长视频一键转化为多个高光短片段，自动适配各大社交平台尺寸。

工作流程 ​

长视频导入 → 精彩片段检测 → 多片段生成 → 尺寸适配 → 批量导出

核心能力 ​

能力说明
精彩片段检测识别欢呼、笑点、高能时刻
多平台适配自动裁剪为 9:16（抖音）/ 4:5（小红书）/ 16:9（B站）
批量导出一次生成多格式多尺寸
话题标签自动生成适合的话题标签

适用场景 ​

  • 📺 直播录屏切片
  • 🎙️ 播客/访谈精华
  • 📈 会议记录摘要
  • 🎮 游戏直播高光

🌐 AI 视频翻译 ​

一键将视频翻译为多语言版本，支持配音和字幕同步。

支持语言 ​

语言类别数量代表语言
中文简/繁普通话、粤语
英语多口音英式、美式、印度
日韩2+日语、韩语
东南亚5+泰语、越南语、印尼语
欧洲15+法、德、西、意、葡
中东5+阿拉伯语、波斯语
总计100+—

核心能力 ​

  • 语音识别 — Whisper 多语言识别（准确率 95%+）
  • 智能翻译 — 保留原意的同时适应目标语言文化
  • 配音合成 — 目标语言自然语音配音
  • 字幕生成 — 目标语言字幕（可嵌入或独立文件）
  • 唇形同步 — （高级功能）口型匹配

适用场景 ​

  • 🌍 内容出海 / 本地化
  • 📚 教育视频翻译
  • 🎬 海外短视频搬运
  • 🤝 跨国企业培训
