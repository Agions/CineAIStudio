## 1️⃣ 菜单栏

| 菜单 | 主要功能 |
|------|----------|
| **文件** | 新建项目、打开、保存、导出、退出 |
| **编辑** | 撤销、重做、剪切、复制、粘贴、删除 |
| **视图** | 全屏预览、面板布局、时间线缩放 |
| **AI 创作** | 剧情分析、AI 解说、智能混剪、独白模式、短视频切片、视频翻译 |
| **导出** | 快速导出、自定义导出、批量导出 |
| **帮助** | 文档、检查更新、关于、反馈问题 |

## 2️⃣ 工具栏

快速访问常用操作按钮：

> 📸 **截图位置**：工具栏各按钮的功能提示（鼠标悬停时显示）

## 3️⃣ 素材面板

> 📸 **截图位置**：素材面板区域，包含项目树、AI 分析结果、字幕轨道

### 项目区

- **视频素材** — 显示所有导入的视频文件
- **音频素材** — 显示背景音乐和配音文件
- **图片素材** — 显示导入的图片文件

### AI 分析区

| 标签 | 说明 |
|------|------|
| 📊 场景 | AI 分析出的场景列表 |
| 🎬 镜头 | AI 检测出的镜头列表 |
| 💬 文案 | 生成的解说词/字幕 |

### 字幕轨道区

显示 AI 生成或手动添加的字幕轨道。

---

## 4️⃣ 预览区域

> 📸 **截图位置**：主预览窗口、播放控制栏

### 预览窗口

- 实时显示当前时间点的视频画面
- 支持全屏预览（双击或按 `F`）
- 右键菜单：截图、复制帧、显示信息

### 播放控制栏

| 按钮 | 说明 |
|------|------|
| ▶️ 播放/暂停 | 空格键 |
| ⏹️ 停止 | 停止并回到起点 |
| ⏮️ 上一帧 | 左箭头 |
| ⏭️ 下一帧 | 右箭头 |
| 🔊 | 音量控制 |
| ⛶ | 全屏切换 |

### 时间码显示

```
当前时间 / 总时长
00:05:23 / 00:30:00
```

## 5️⃣ 时间线面板

> 📸 **截图位置**：多轨道时间线，包含缩放控制和吸附开关

### 轨道结构

| 轨道 | 说明 |
|------|------|
| V1 / V2 | 视频轨道（V1 主轨，V2 叠加） |
| A1 / A2 | 音频轨道（A1 配音，A2 背景音乐） |
| S1 | 字幕轨道 |

### 时间线操作

| 操作 | 方法 |
|------|------|
| 移动播放头 | 点击时间尺或拖动播放头 |
| 缩放 | 滚轮缩放 / 缩放滑块 |
| 平移 | 中键拖动 / 方向键 |
| 选择片段 | 单击片段 |
| 移动片段 | 拖动片段 |
| 裁剪片段 | 拖动片段边缘 |
| 分割片段 | 按 `S` 在播放头位置分割 |
| 删除片段 | 选中后按 `Delete` |

### 缩放控制

```
缩小 ──────●──────────── 放大
```

### 吸附控制

- 📌 **开** — 自动吸附到剪辑点、标记点
- 📌 **关** — 自由拖动

## 界面定制

### 主题切换

右上角 ☀️/🌙 按钮切换浅色/深色主题（深色为默认）。

### 面板布局

- **拖拽调整** — 拖动面板分隔线调整大小
- **折叠面板** — 双击面板标题栏折叠/展开
- **重置布局** — 视图 → 重置面板布局

### 时间线定制

- 添加/删除轨道
- 调整轨道高度
- 更改轨道颜色
- 波形显示/隐藏

::: tip 💡 建议
第一次使用时，建议先熟悉时间线的基本操作（移动片段、裁剪、分割），这是所有剪辑工作的基础。
:::
