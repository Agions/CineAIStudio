## 面板说明

### 1. 素材面板

| 区域 | 功能 |
|------|------|
| 项目 | 管理导入的素材 |
| AI 分析 | 显示 AI 分析结果 |
| 字幕 | 字幕轨道管理 |
| 音频 | 音频轨道管理 |

### 2. 预览区域

- 实时预览剪辑效果
- 播放控制：播放/暂停/停止
- 时间码显示
- 全屏预览

### 3. 时间线面板

- 多轨道编辑
- 缩放和平移
- 剪辑点拖拽
- 波形显示

### 4. 状态栏

- 系统资源使用情况
- AI 服务状态
- 当前操作提示
