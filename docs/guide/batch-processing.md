# 批量处理指南

VideoForge 的批量处理功能允许你同时对多个视频进行 AI 处理，大大提高工作效率。

## 适用场景

- 📺 **直播切片** — 将多场直播录屏转化为高光片段
- 🎬 **系列视频** — 批量处理课程视频、教程系列
- 📱 **多平台分发** — 同一素材批量导出不同格式/分辨率
- 🌍 **本地化翻译** — 一键生成多语言版本

---

## 打开批量处理面板

入口：**AI 创作 → 批量处理**

或使用快捷键：`Ctrl+Shift+B`（Windows）/ `⌘⇧B`（macOS）

---

## 添加处理任务

### 方法一：手动添加

1. 点击 **+ 添加任务**
2. 选择视频文件
3. 选择处理模式
4. 配置参数
5. 点击 **确认**

### 方法二：批量导入

支持一次性导入整个文件夹：

1. 点击 **批量导入**
2. 选择包含视频文件的文件夹
3. 设置统一的处理模式和参数
4. 点击 **开始排队**

### 方法三：拖拽添加

直接将视频文件或文件夹拖入批量处理面板。

---

## 任务配置

### 基础配置

| 参数 | 说明 |
|------|------|
| **处理模式** | 选择 AI 创作模式 |
| **输出目录** | 统一设置导出路径 |
| **命名规则** | 支持变量：`{filename}`、`{index}`、`{date}` |

### 高级配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 并行数 | 同时处理的任务数 | 1 |
| 失败重试 | 失败后自动重试次数 | 2 |
| 自动导出 | 处理完成后自动导出 | ✅ |
| 完成后通知 | 全部完成后发送系统通知 | ✅ |

## 队列管理

### 任务状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 等待中 | ⏳ | 排队等待处理 |
| 正在处理 | 🔄 | 正在 AI 处理 |
| 导出中 | 📦 | 正在导出文件 |
| 已完成 | ✅ | 处理成功 |
| 已失败 | ❌ | 处理失败（可重试） |
| 已暂停 | ⏸️ | 用户暂停 |

### 队列操作

- **暂停全部** — 暂停队列中所有任务
- **恢复全部** — 恢复已暂停的任务
- **清空已完成** — 移除已完成的任务记录
- **清空全部** — 清空整个队列（谨慎操作）

### 单任务操作

- **▶️ 开始** — 立即开始处理此任务
- **⏸️ 暂停** — 暂停此任务
- **🔄 重试** — 重新处理失败的任务
- **🗑️ 移除** — 从队列中移除此任务
- **📝 编辑** — 修改任务参数

## 模板与预设

### 保存为模板

将常用配置保存为模板：

1. 配置好处理参数
2. 点击 **保存为模板**
3. 输入模板名称
4. 下次使用时直接选择模板

### 预设模板

VideoForge 内置以下模板：

| 模板名称 | 适用场景 | 模式 | 并行数 |
|----------|----------|------|--------|
| **直播高光切片** | 直播录屏 | 短视频切片 | 2 |
| **课程批量解说** | 在线课程 | AI 解说 | 1 |
| **系列混剪** | 系列素材 | 智能混剪 | 2 |
| **多语言分发** | 出海内容 | 视频翻译 | 1 |

## 故障排除

### 任务一直处于"等待中"

检查队列是否被暂停，或并行数是否为 0。

### 任务失败率高

常见原因：
1. 源视频格式不兼容（尝试转为 MP4）
2. 磁盘空间不足
3. API 请求超限（降低并行数）
4. 网络不稳定

### 内存不足

- 降低并行数（建议设为 1）
- 关闭其他占用内存的程序
- 使用较低分辨率预览

## 快捷操作

| 操作 | Windows | macOS |
|------|---------|-------|
| 打开批量处理 | `Ctrl+Shift+B` | `⌘⇧B` |
| 添加任务 | `Ctrl+N` | `⌘N` |
| 暂停全部 | `Ctrl+P` | `⌘P` |
| 清空已完成 | `Ctrl+Shift+C` | `⌘⇧C` |

::: tip 💡 提示
对于大型批量任务（如 100+ 视频），建议在夜间无人使用时运行，并设置"完成后通知"以便第二天查看结果。
:::

::: warning ⚠️ 注意
批量处理会消耗大量系统资源。建议在处理期间避免进行其他高负载操作（如游戏、大文件拷贝等）。
:::
