🚀 开始使用 ​

安装 NARRAFILM ​

📦下载安装包
访问 [GitHub Releases](https://github.com/Agions/NARRAFILM/releases/latest) 下载对应平台的最新版本。
🤖配置 AI
只需一个 API Key 即可启动（推荐 DeepSeek），场景理解 + 解说生成全部本地可跑。

导入视频 → 选择情感风格 → 一键生成配音解说 → 预览并导出。
