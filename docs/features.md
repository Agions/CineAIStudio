# 功能介绍

Narrafiilm 提供六大 AI 创作模式，覆盖从影视解说到社交媒体分发的全场景。

---

## 🎬 剧情分析

智能分析视频的叙事结构、情感曲线和剧情发展，自动生成基于故事的剪辑建议。

### 核心能力

- **场景分割** — 自动识别场景边界，生成场景列表
- **情感分析** — 分析每个场景的情感倾向（喜怒哀乐）
- **叙事提取** — 提取关键情节和高光时刻
- **剪辑建议** — 基于叙事结构推荐剪辑点

### 适用场景

- 📽️ 电影解说 / 影评视频
- 📷 Vlog 整理与叙事重构
- 🎞️ 纪录片素材整理
- 📚 课程视频结构化

### 界面位置

> 📸 界面截图位置：`app/ui/main/pages/ai_video_creator.py` → 模式选择器 → 剧情分析模式

## 🎙️ AI 视频解说

将原始视频自动转化为带 AI 配音和动态字幕的解说视频。

### 工作流程

```
视频导入 → 场景分析 → 文案生成 → 配音合成 → 字幕制作 → 视频合成 → 导出
```

### 核心能力

| 步骤 | 技术 | 说明 |
|------|------|------|
| 场景分析 | PySceneDetect / Gemini | 识别视频结构和内容 |
| 文案生成 | GPT-4o / Claude | 生成自然流畅的解说词 |
| 配音合成 | Edge TTS / OpenAI TTS | 支持多音色、多语言 |
| 字幕生成 | Whisper | 语音转文字 + 时间轴 |
| 视频合成 | FFmpeg | 音画合成 + 字幕叠加 |

### 适用场景

- 🎓 在线课程录制
- 📦 产品介绍视频
- 📰 新闻/纪录片配音
- 🎮 游戏解说

### 界面位置

> 📸 界面截图位置：`app/ui/main/pages/ai_video_creator.py` → 模式选择器 → AI 解说模式

### 参数配置

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 解说风格 | 专业 / 轻松 / 新闻 | 专业 |
| 语速 | 0.8x ~ 1.5x | 1.0x |
| 音调 | 低沉 / 正常 / 高亢 | 正常 |
| 字幕样式 | 电影 / 卡片 / 底部 | 电影 |

## 🎵 AI 智能混剪

将多个素材智能混剪成节奏感强的视频，自动完成节拍匹配和转场合成。

### 工作流程

```
多素材导入 → 节拍检测 (BPM) → 素材排序 → 剪辑点生成 → 转场合成 → 导出
```

### 核心能力

- **BPM 检测** — 自动检测音频/视频的节拍，精确到毫秒
- **智能排序** — 根据节拍和内容关联性自动排序素材
- **自动转场** — AI 生成适合节奏的转场效果（叠化、闪白、节奏切）
- **多轨混音** — 支持背景音乐 + 原音混合

### 适用场景

- 🎵 音乐 MV / 混剪视频
- ⚽ 体育精彩集锦
- 🎮 游戏高光时刻
- 📸 旅行/生活照片集

### 界面位置

> 📸 界面截图位置：`app/ui/main/pages/ai_video_creator.py` → 模式选择器 → 智能混剪模式

## 🎭 AI 第一人称独白

分析视频画面情感，生成对应的情感独白，配以电影级字幕风格。

### 核心能力

| 能力 | 说明 |
|------|------|
| 情感检测 | 分析画面中的人物表情、场景氛围 |
| 独白生成 | 生成符合情感的内心独白 |
| 情感配音 | 使用更柔和/更有感染力的声音 |
| 电影字幕 | 专业的电影感字幕样式 |

### 字幕样式

```
┌──────────────────────────────────────────────┐
│                                              │
│         我一直相信，                          │
│         每一次相遇都有它的意义。               │
│                                              │
│              —— 2024年夏，云南                 │
└──────────────────────────────────────────────┘
```

### 适用场景

- 💭 情感纪录片
- 🌍 旅行纪录片
- 🎬 Vlog 电影感重构
- 📖 人物专题片

## 📱 AI 短视频切片

长视频一键转化为多个高光短片段，自动适配各大社交平台尺寸。

### 工作流程

```
长视频导入 → 精彩片段检测 → 多片段生成 → 尺寸适配 → 批量导出
```

### 核心能力

| 能力 | 说明 |
|------|------|
| 精彩片段检测 | 识别欢呼、笑点、高能时刻 |
| 多平台适配 | 自动裁剪为 9:16（抖音）/ 4:5（小红书）/ 16:9（B站） |
| 批量导出 | 一次生成多格式多尺寸 |
| 话题标签 | 自动生成适合的话题标签 |

### 适用场景

- 📺 直播录屏切片
- 🎙️ 播客/访谈精华
- 📈 会议记录摘要
- 🎮 游戏直播高光

## 🌐 AI 视频翻译

一键将视频翻译为多语言版本，支持配音和字幕同步。

### 支持语言

| 语言类别 | 数量 | 代表语言 |
|----------|------|----------|
| 中文 | 简/繁 | 普通话、粤语 |
| 英语 | 多口音 | 英式、美式、印度 |
| 日韩 | 2+ | 日语、韩语 |
| 东南亚 | 5+ | 泰语、越南语、印尼语 |
| 欧洲 | 15+ | 法、德、西、意、葡 |
| 中东 | 5+ | 阿拉伯语、波斯语 |
| **总计** | **100+** | — |

### 核心能力

- **语音识别** — Whisper 多语言识别（准确率 95%+）
- **智能翻译** — 保留原意的同时适应目标语言文化
- **配音合成** — 目标语言自然语音配音
- **字幕生成** — 目标语言字幕（可嵌入或独立文件）
- **唇形同步** — （高级功能）口型匹配

### 适用场景

- 🌍 内容出海 / 本地化
- 📚 教育视频翻译
- 🎬 海外短视频搬运
- 🤝 跨国企业培训

## 支持的 AI 模型

Narrafiilm 支持多种 AI 模型，可根据场景和预算灵活切换：

| 提供商 | 模型 | 上下文 | 适用场景 |
|--------|------|--------|----------|
| **OpenAI** | GPT-5.4 | 128K | 剧情分析、脚本生成 |
| **Anthropic** | Claude Sonnet 4.6 | 200K | 长文本分析、安全性要求高 |
| **Google** | Gemini 3.1 Pro | 1M | 视频理解、超长文本 |
| **DeepSeek** | V3.2 | 128K | 翻译、日常任务（性价比最高） |
| **阿里云** | Qwen 2.5-Max | 128K | 中文创作 |
| **智谱** | GLM-5 | 128K | 中文创作 |
| **Kimi** | K2.5 | 200K+ | 超长上下文分析 |
| **Ollama** | 本地模型 | 可配置 | 完全私有离线 |

::: tip 💡 模型选择建议
- **免费推荐**：DeepSeek V3.2 + Edge TTS（完全免费，效果优秀）
- **性价比推荐**：DeepSeek V3.2 + OpenAI TTS
- **最强能力**：GPT-5.4 + OpenAI TTS
- **中文创作**：Qwen 2.5-Max / GLM-5
- **隐私敏感**：Ollama 本地部署
:::

## 技术特性

### GPU 加速

Narrafiilm 支持 GPU 加速，显著提升 AI 处理速度：

| 功能 | CPU | GPU (CUDA) | 加速比 |
|------|-----|------------|--------|
| 语音识别 (Faster-Whisper) | ~3x 实时 | ~30x 实时 | 10x |
| 视频分析 (Gemini) | - | GPU 加速 | - |

**自动检测**：系统会自动检测 CUDA 可用性，有 GPU 时自动启用加速，无 GPU 时自动回退到 CPU。

### Nuitka 打包（可选）

可选使用 Nuitka 替代 PyInstaller 打包：

| 指标 | PyInstaller | Nuitka |
|------|--------------|--------|
| 包体积 | ~500MB | ~300MB |
| 启动时间 | ~5s | ~2s |
| 编译时间 | 快 | 慢 (3-5分钟) |

```bash
# 使用 Nuitka 构建
make build:nuitka
```
