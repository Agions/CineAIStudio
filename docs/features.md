# 核心功能

ClipFlowCut 提供三大 AI 创作模式，满足不同场景的视频创作需求。

## 🎙️ AI 视频解说

将原始视频自动转化为带 AI 配音和动态字幕的解说视频。

**工作流：**
1. 场景分析 - 自动分割视频场景
2. 文案生成 - AI 生成解说词
3. AI 配音 - 使用 Edge TTS 生成语音
4. 字幕合成 - 动态字幕匹配

**适用场景：**
- 电影解说
- 纪录片配音
- 产品介绍

## 🎵 AI 视频混剪

将多个素材智能混剪成节奏感强的视频。

**工作流：**
1. 素材分析 - 分析每个片段内容
2. BPM 检测 - 检测背景音乐节拍
3. 智能剪辑 - 根据节奏自动剪辑
4. 转场合成 - 添加转场效果

**适用场景：**
- 短视频混剪
- 音乐 MV
- 精彩集锦

## 🎭 AI 第一人称独白

分析视频画面情感，生成对应的情感独白。

**工作流：**
1. 情感分析 - 分析画面情感
2. 独白生成 - 生成匹配的情感文案
3. 配音合成 - 电影级别配音
4. 电影字幕 - 添加电影风格字幕

**适用场景：**
- 情感纪录片
- 内心独白
- 电影感视频
