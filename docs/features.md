# 功能介绍

Voxplore v4.0 核心定位：**多视频智能合并解说** — 批量上传视频 → AI 智能分组 → 提取第一人称片段 → 叙事完整选段 → 模块化成品输出，短剧/影视解说一步到位。

---

## 核心工作流

```
多视频上传（文件夹 / Ctrl多选）
    │
    ▼
┌─────────────────────────────────────────────────┐

│  Qwen2.5-VL 视觉 embedding（0.7）               │
│  声纹识别 音频 embedding（0.3）                  │
│  → 避免同一人重复生成解说                        │
└─────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────┐

│  逐帧分析画面 → 判断"我"的主体视角               │

└─────────────────────────────────────────────────┘
    │
    ▼

│  叙事完整 + 情感峰值 双重选段                    │

    │

    │
    ▼
┌─────────────────────────────────────────────────┐
│  解说生成 + 配音合成                             │
│  预设风格（7种）+ 角色设定                       │

```

## 多视频智能合并

### 上传方式

| 方式 | 说明 |
|------|------|
| **文件夹选择** | 选一个文件夹，自动扫描所有视频（mp4/mov/avi/webm）|
| **Ctrl 多选** | 按住 Ctrl 一次选中多个视频，统一加入队列 |
| **混合模式** | 文件夹 + 多选，AI 自动去重合并 |

### 智能分组算法

**视觉 + 音频 混合方案：**

| 维度 | 技术 | 权重 |
|------|------|------|
| 视觉 embedding | Qwen2.5-VL | 0.7（主导）|
| 音频声纹 | 声纹识别 | 0.3（辅助修正）|

**分组结果展示：**
- 每组显示视频缩略图网格（3列）
- 置信度颜色指示：>80% 绿 / 60-80% 黄 / <60% 红
- 支持手动合并/拆分分组
- 支持新建/删除分组

## 第一人称片段提取

### 提取逻辑

1. **逐帧采样**：采样关键帧，分析每帧画面
2. **视角判断**：Qwen2.5-VL 判断"我"的主体视角
3. **连续性过滤**：提取时序连续的第一人称片段
4. **时长过滤**：保留 9–60 秒的片段（短视频黄金区间）

### 片段信息

每个候选片段包含：
- `start_time` / `end_time`：起止时间（秒）
- `confidence`：第一人称置信度（0.0–1.0）
- `description`：片段画面描述

## 情感峰值检测

### 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 画面信息密度 | 0.6 | 复杂度/动作密度/视觉刺激 |
| 音频情绪起伏 | 0.4 | 语调变化/情感强度 |

### 峰值原因

- 🔥 **高复杂度**：场景信息量大，细节丰富
- 😱 **强情绪**：语调强烈变化，情绪波动大
- ⚡ **动作密度**：画面动作密集，高光时刻

## 叙事完整选段

### 选段策略

| 策略 | 说明 |
|------|------|
| **叙事完整优先** | 每个片段必须有头有尾，保证信息完整 |
| **情感峰值驱动** | 情感峰值片段加权优先入选 |
| **混合策略（默认）** | 叙事 0.7 + 情感 0.3，综合评分选最优 |

### 目标时长

| 平台 | 目标时长 |
|------|----------|
| 抖音 | 15–30 秒 |
| B站 | 3–10 分钟 |
| 小红书 | 30s–2 分钟 |

系统按目标平台自动调整，也支持手动设置。

## 解说风格 + 角色设定

### 7 种预设风格

| 风格 | 适用场景 | 解说语气 |
|------|----------|----------|
| **治愈** | 旅行、自然风光、生活记录 | 温暖、放松、随性 |
| **悬疑** | 惊悚、探秘、推理向内容 | 低沉、留白、制造紧张感 |
| **励志** | 挑战、成长、努力过程 | 坚定、有力、积极 |
| **怀旧** | 回忆、老物件、时光流逝 | 柔和、感慨、余韵悠长 |
| **浪漫** | 情感、约会、美好时刻 | 细腻、温柔、充满画面感 |
| **幽默** | 有趣、搞笑、反转内容 | 活泼、俏皮、制造笑点 |
| **纪录片** | 知识、科普、深度解读 | 专业、冷静、逻辑清晰 |

### 角色设定

| 参数 | 说明 |
|------|------|
| **主角名称** | 解说中的"我"的名字 |
| **职业设定** | 医生/律师/教师/销售/自由职业/... |
| **性格特征** | 内向/外向/理性/感性/幽默/沉稳/... |
| **口头禅** | 可自定义关键词，AI 在解说中自然融入 |

## 模块化成品

### 成品类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **合并版** | 所有选段按叙事顺序拼接成一条完整视频 | 抖音/B站完整版发布 |
| **高光片段** | 最精彩的片段单独导出 | 短视频平台分发最大化 |
| **剪映草稿** | 原生 `.draft.json`，导入剪映继续精剪 | 专业后期编辑 |

### 导出格式

| 格式 | 编码 | 分辨率 |
|------|------|--------|
| MP4（H.264）| 默认，兼容性最好 | 保持原片 / 1080p / 720p |
| MP4（H.265）| 体积小约 40% | 同上 |
| 剪映草稿 JSON | 原生格式 | — |

## 精准字幕

基于 TTS word-level timing 的音字同步，字幕精确到每个字的起止时间（50ms 以内精度）。

### 字幕样式

| 样式 | 效果 |
|------|------|
| **电影黑底白字** | 底部居中，黑底白字，带淡淡描边 |
| **透明覆盖** | 底部居中，无背景，文字描边 |
| **卡片式** | 全宽底栏，半透明背景 |
| **顶部字幕** | 用于说明画面外事件 |

## 支持的视频格式

| 格式 | 扩展名 | 推荐度 |
|------|--------|--------|
| MP4 | `.mp4` | 高，兼容性最好 |
| MOV | `.mov` | 高，macOS 录制格式 |
| MKV | `.mkv` | 中，支持多音轨 |
| AVI | `.avi` | 低，体积较大 |
| WebM | `.webm` | 低，部分编码有限制 |

系统自动检测 CUDA，可用时自动启用 GPU 加速。
