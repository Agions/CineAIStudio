快速开始 ​

⏱️ 预计时间：5 分钟（已安装应用者 2 分钟）

系统要求 ​

项目最低要求推荐
操作系统Windows 10+ / macOS 11+ / Ubuntu 20.04+Windows 11 / macOS 13+
内存8 GB16 GB
显卡—NVIDIA 4GB+（CUDA 加速）
存储2 GB10 GB（包含模型缓存）
网络用于调用 AI API—

下载与安装 ​

###方式一：安装包（推荐）

  1. 打开 GitHub Releases
  2. 下载对应平台版本：
平台文件名
WindowsNarrafiilm-Setup-x.x.x.exe
macOSNarrafiilm-x.x.x.dmg
LinuxNarrafiilm-x.x.x.AppImage
  1. 运行安装包，按提示完成

方式二：Homebrew（macOS / Linux） ​

bash
brew install agions/tap/narrafiilm

方式三：源码安装 ​

bash
git clone https://github.com/Agions/Narrafiilm.git
cd narrafiilm
python -m venv venv && source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app/main.py

⚠️ 源码安装需要自行安装 FFmpeg。

配置 API Key ​

Narrafiilm 需要一个 AI API Key 来生成解说词。DeepSeek 性价比最高，每月约 $1 可处理数十个视频。

获取 DeepSeek API Key ​

  1. 打开 platform.deepseek.com
  2. 注册并登录
  3. 进入 API Keys → Create API Key
  4. 复制生成的 Key（格式：sk-...）

填入应用 ​

启动 Narrafiilm → 设置 → AI 配置 → 粘贴 Key → 保存

或直接编辑项目根目录的 .env 文件：

env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

验证连接 ​

进入 设置 → AI 配置，点击 连接测试。出现 ✅ 即表示配置成功。

创建第一个解说视频（3 步） ​

第 1 步：新建项目 ​

启动应用 → 点击 新建项目 → 选择保存位置 → 输入项目名称

第 2 步：导入视频 ​

  • 点击 导入素材 按钮，或直接将视频文件拖入窗口
  • 支持格式：MP4 / MOV / AVI / MKV / WebM

第 3 步：生成解说 ​

  1. 确认视频已加载
  2. 选择情感风格：治愈 / 悬疑 / 励志 / 怀旧 / 浪漫
  3. 选择配音音色：默认 XiaoXiao（女声）
  4. 点击 开始创作

等待 AI 完成处理（通常 1–3 分钟，取决于视频长度和 API 速度），预览结果后点击 导出。

常见问题 ​

提示 "FFmpeg not found" ​

FFmpeg 是视频处理的核心依赖，未安装会导致所有视频操作失败。

bash
# macOS
brew install ffmpeg

# Ubuntu / Debian
sudo apt install ffmpeg

# Windows
# 下载 https://www.gyan.dev/ffmpeg/builds/ ，解压后将 bin 目录加入 PATH

安装后重启应用。

AI 功能报错 401 / 403 ​

API Key 无效或额度用尽。进入 platform.deepseek.com 检查 Key 状态和账户余额。

处理速度很慢 ​

  • 确认网络连接稳定（调用外部 API）
  • 有 NVIDIA 显卡时自动启用 CUDA 加速
  • 减少抽帧密度可提速（设置 → 场景理解 → 抽帧间隔）

显存不足（OOM） ​

GPU 模式对显存要求较高。进入 设置 → AI 配置，关闭 启用 GPU 加速，回退到 CPU 模式。

无 DISPLAY 环境运行静默退出 ​

Linux 无头环境（SSH / 服务器）下，应用会自动设置 QT_QPA_PLATFORM=offscreen，无需额外配置。
