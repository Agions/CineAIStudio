导出格式详解 ​

MP4（直接输出） ​

最常用的输出格式，兼容性最好。

编码选项 ​

参数选项默认说明
编码H.264 / H.265H.264H.264 兼容性最好，H.265 体积小约 40%
分辨率保持原片 / 1080p / 720p / 480p保持原片下采样可大幅减少体积
帧率保持原片 / 60 / 30 / 24保持原片—
码率自动 / 5Mbps / 10Mbps / 20Mbps自动自动根据内容自适应
音轨配音+原音 / 仅配音 / 仅原音仅配音—

推荐配置 ​

用途配置
微信 / QQ 分享H.264 / 720p / 10Mbps
B站 / YouTube 上传H.264 / 1080p / 自动码率
存档备份H.265 / 保持原片 / 自动码率
移动端查看H.264 / 480p / 5Mbps

剪映草稿 JSON ​

导出原生 .draft.json 文件，可直接导入剪映继续精剪。

导入剪映 ​

  1. 导出完成后，打开剪映
  2. 点击 导入 → 选择导出的 .draft.json 文件
  3. 草稿自动加载，包含：
    • 时间轴和分割点
    • AI 配音轨道
    • ASS 字幕轨道（可继续编辑文字和时间）

注意事项 ​

  • 使用最新版本剪映（较老版本可能不兼容）
  • 草稿路径避免中文字符
  • 剪映国际版（CapCut）部分功能不兼容中文路径草稿

ASS 字幕单独导出 ​

Narrafiilm 可以单独导出 ASS 字幕文件，不含视频：

  • .ass 文件：完整样式（位置/颜色/动画）
  • .srt 文件：纯文本时间轴（通用格式）

适用于：

  • 用 Aegisub 等专业字幕工具进一步调轴
  • 导入 Premiere / DaVinci Resolve 继续后期
  • 单独分享字幕文件

输出目录 ​

默认输出到：项目目录 / output / [日期时间戳] /

output/
└── 2026-04-05_14-30-00/
    ├── video_final.mp4          # 成品视频
    ├── video_draft.json         # 剪映草稿（如选择）
    ├── subtitle.ass             # ASS 字幕
    ├── subtitle.srt             # SRT 字幕
    ├── narration.mp3            # 配音音频（可选）
    └── script.txt               # 解说稿原文
