欢迎使用 Narrafiilm ​

Narrafiilm 是专业的 AI 第一人称视频解说 桌面应用。上传任意视频，AI 自动代入画面主角视角，生成"我"的口吻配音解说 + 电影级字幕 + 精准同步 ASS 字幕。

快速开始 ​

  • 5 分钟快速开始 — 最简化的上手指引
  • 安装配置 — 各平台详细安装步骤
  • 功能介绍 — 了解全部能力

核心工作流 ​

步骤说明
📹 上传视频支持 MP4 / MOV / AVI / MKV / WebM
🧠 AI 场景理解Qwen2.5-VL 逐帧分析画面，识别主角视角
✍️ 生成解说稿DeepSeek-V3 代入"我"的第一人称视角
🎙️ 情感配音Edge-TTS / F5-TTS 高保真合成
✨ 合成输出MP4 成品 / 原生剪映草稿 JSON

主要特性 ​

特性说明
第一人称代入Qwen2.5-VL 识别画面主角，DeepSeek-V3 代入"我"视角
5 种情感风格治愈 / 悬疑 / 励志 / 怀旧 / 浪漫
精准 ASS 字幕TTS word-level timing，50ms 以内同步精度
多格式导出H.264/H.265 MP4 或原生剪映草稿 JSON
