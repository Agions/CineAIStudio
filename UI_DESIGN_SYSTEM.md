# CineFlow UI 设计系统

## 概述
统一的设计系统，为 CineFlow 提供一致的视觉体验。

## 颜色系统

### 主色 (Primary)
- Primary: `#2196F3` (蓝色)
- Primary Dark: `#1976D2`
- Primary Light: `#64B5F6`

### 辅助色 (Secondary)
- Secondary: `#9C27B0` (紫色)
- Accent: `#FF4081` (粉红)

### 功能色
- Success: `#4CAF50` (绿色)
- Warning: `#FF9800` (橙色)
- Error: `#F44336` (红色)
- Info: `#2196F3` (蓝色)

### 背景色
- Background Dark: `#1E1E1E`
- Surface Dark: `#2D2D2D`
- Card Dark: `#2D2D2D`
- Border Dark: `#404040`

## 字体系统
- 标题: 16px, Bold
- 正文: 14px, Regular
- 小字: 12px, Regular

## 间距系统
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
