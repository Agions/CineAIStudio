## 🎨 设计理念

### 1. 色彩系统重构

```
主色调: 渐变紫蓝 (#6366F1 → #8B5CF6)
强调色: 青色 (#06B6D4)
成功: 翠绿 (#10B981)
警告: 琥珀 (#F59E0B)
错误: 玫瑰 (#EF4444)

背景层次:
- 最深: #0A0A0F (主背景)
- 深色: #12121A (卡片背景)
- 中层: #1A1A24 (悬浮层)
- 浅层: #22222E (交互层)
```

### 2. 玻璃拟态 (Glassmorphism)

```css
/* 卡片 */
background: rgba(26, 26, 36, 0.8);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.08);

/* 悬浮态 */
background: rgba(36, 36, 50, 0.9);
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
```

### 3. 微动效

| 元素 | 动效 |
|------|------|
| 按钮悬停 | scale(1.02), 150ms ease |
| 卡片悬浮 | translateY(-4px), 200ms |
| 页面切换 | fade + slide, 300ms |
| 加载 | 脉冲发光 |
| 点击 | 波纹扩散 |

## 📐 布局优化

### 主窗口结构

```
┌─────────────────────────────────────────────────────────┐
│  顶部栏 (48px) - Logo | 搜索 | 用户                    │
├────────┬────────────────────────────────────────────────┤
│        │                                                │
│  侧边栏 │              主内容区                          │
│  (240px)│              (flex)                           │
│        │                                                │
│  导航   │  ┌──────────────────────────────────────┐   │
│  快捷键 │  │  工具栏 (44px)                       │   │
│        │  ├──────────────────────────────────────┤   │
│        │  │                                      │   │
│        │  │         画布 / 时间轴                 │   │
│        │  │                                      │   │
│        │  └──────────────────────────────────────┘   │
├────────┴────────────────────────────────────────────────┤
│  状态栏 (28px) - 进度 | 快捷操作 | 时间               │
└─────────────────────────────────────────────────────────┘
```

### 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| 大屏 | ≥1440px | 完整布局 |
| 中屏 | 1024-1439 | 收起侧边栏 |
| 小屏 | 768-1023 | 底部导航 |
| 移动 | <768 | 简化模式 |

## 🔧 实现计划

### 阶段 1: 基础改造 (1-2天)

1. **色彩变量统一**
   - 在 `theme_manager.py` 定义完整色彩系统
   - CSS 变量化

2. **背景重构**
   - 添加渐变背景
   - 噪点纹理

3. **卡片玻璃拟态**
   - 重写 `modern.qss`

### 阶段 2: 交互增强 (2-3天)

1. **动效系统**
   - 定义动画 keyframes
   - 按钮/卡片动效

2. **加载状态**
   - 骨架屏
   - 进度动画

3. **页面转场**
   - QStackedWidget 动画

### 阶段 3: 品牌升级 (1-2天)

1. **启动页**
   - 动画 Logo

2. **空状态**
   - 插画 + 引导

3. **图标系统**
   - 统一图标风格
