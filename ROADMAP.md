## 🤝 贡献指南

### 如何参与?

1. **反馈问题**: 在 GitHub Issues 中报告 Bug
2. **提出建议**: 提出新功能或改进建议
3. **贡献代码**:
   - Fork 项目
   - 创建功能分支
   - 编写测试
   - 提交 Pull Request

### 代码规范

- 遵循 PEP 8
- 编写文档字符串
- 添加单元测试
- 提交信息规范 (参考 DEVELOPER.md)

### 测试要求

- 单元测试覆盖率 >70%
- 集成测试通过
- 无 linting 警告
