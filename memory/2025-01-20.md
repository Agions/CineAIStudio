### 🎯 功能特性

#### 多Agent协同
```
Director → 制定计划 → 分配任务
    ↓
Editor/Colorist/Sound/VFX (并行)
    ↓
Reviewer → 质量检查
    ↓
Director → 整合输出
```

#### 国产大模型
| Agent | 模型 | 能力 |
|-------|------|------|
| Director | DeepSeek-V3 | 规划推理 |
| Editor | Kimi K2.5 | 256K上下文 |
| Colorist | Kimi K2.5 | 视觉理解 |
| Sound | Qwen 2.5 | 音频多模态 |
| VFX | Kimi K2.5 | 画面理解 |
| Reviewer | DeepSeek-Coder | 代码化检查 |

#### 跨平台支持
- Windows 10/11 (.exe)
- macOS 10.15+ (.app)
- 内置FFmpeg
- 自动平台检测
