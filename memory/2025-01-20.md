### 🎯 功能特性

#### 多Agent协同
```
Director → 制定计划 → 分配任务
    ↓
Editor/Colorist/Sound/VFX (并行)
    ↓
Reviewer → 质量检查
    ↓
Director → 整合输出
```

### ✅ 多Agent任务调度系统 (新增)

#### TaskScheduler
- 优先级队列 (5级)
- 依赖管理
- 并行执行控制
- 超时和重试
- 工作流支持

#### VideoEditingWorkflow
- 标准剪辑工作流
- 并行剪辑工作流
- AI解说视频工作流
- 依赖编排

**使用示例:**
```python
scheduler = TaskScheduler(agent_manager)
await scheduler.start()

workflow = VideoEditingWorkflow(scheduler)
instance_id = await workflow.start(
    video_path='/path/to/video.mp4',
    style='parallel'  # 并行模式
)
```
